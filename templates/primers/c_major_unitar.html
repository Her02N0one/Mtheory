Module 2 · Single String — C Major, One String at a Time

Forget the boxes — the scale is a ruler

Every box shape you've learned is just the major scale folded across six strings. Unfold it onto a single string and the scale becomes pure horizontal distance: the same W–W–H–W–W–W–H formula, measured in frets. Two frets for a whole step, one fret for a half step. Nothing else to track — no string crossings, no shape jumps.

W = 2 frets · H = 1 fret (amber). The two H steps — E→F and B→C — are the only places adjacent notes sit one fret apart.

Read it as fret distance

Start on any C and count frets up the same string. The half steps are the two spots where the next note is just one fret away — every other step skips a fret:

Fret offset from the root C. E→F (+4→+5) and B→C (+11→+12) are the adjacent-fret pairs. The octave lands exactly 12 frets up.

The same shape on every string

Because the gap pattern is fixed, this fret ruler is identical no matter which string you walk it on — only the starting fret moves. Learn the distance once and you can play C major along the low E, A, D, G, B, or high e string without ever leaving it.

What this drill builds

  • Tempo & spacing — you'll feel the two 1-fret half steps break the steady 2-fret rhythm. That irregularity is the major scale.
  • Position independence — once the neck stops being a grid of boxes, you can slide a melody up a single string and stay in key.
  • Ear anchoring — ascending then descending the same line locks the sound of each degree against the root.

One string. One formula. Walk it up, walk it back down, and let the half-step connectors on the fretboard show you exactly where the tension lives.
